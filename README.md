# Magisches-Quadrat

Bei dieser Aufgabe war es das Ziel, eine statische Methode static boolean isMagicSquare(int[][] square) zu implementieren, die prüft, ob ein übergebenes, bereits quadratisches int-Array ein magisches Quadrat ist. Dazu sollte die Methode kontrollieren, ob die Summe aller Zeilen, aller Spalten und beider Diagonalen jeweils denselben Wert haben und entsprechend true (magisches Quadrat) oder false zurückgeben.
